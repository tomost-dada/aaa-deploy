$ARGUMENTS 주차를 통합 분석해줘.

## 실행 절차

### 1. 데이터 수집
- `00_주차별미션/Week_0$ARGUMENTS/` 의 모든 멤버 파일 읽기
- `01_개념/` 의 모든 파일 읽기
- `02_인사이트/` 의 모든 파일 읽기
- 파일 크기 10바이트 이하인 파일은 미제출로 간주하고 건너뛰기

### 2. 제출 현황 집계
각 멤버(다다, 비비안, 띵끄, 잽, 흐민, 에밀리, 다니, 오웬)별로:
- 미션 제출 여부
- 개념 작성 여부
- 인사이트 작성 여부

### 3. 통합 분석
수집된 모든 내용을 종합하여:
- **주요 키워드** 추출 (빈도 순)
- **공통 관심사** 파악 (3명 이상 언급한 주제)
- **독특한 시각** 발견 (1명만 언급한 흥미로운 관점)
- **프로젝트 아이디어 시드** 도출
- **스킬/플러그인 제안** 도출: Claude Code 스킬, Obsidian 플러그인, CLI 도구 아이디어 (난이도, 관련 개념, 추천 멤버 포함)

### 4. 결과 파일 생성
- `_분석/주차별분석/Week_0$ARGUMENTS_분석.md` 생성 (`_templates/분석_Week_0N.md` 형식)
- 각 멤버별 `_분석/멤버별분석/닉네임_프로필.md` 업데이트 (누적)

### 5. 현황 업데이트
- `_현황/진행현황.md` 업데이트
  - 출석 현황 테이블: 주차별 O/X + 결석 수
  - 과제 현황 테이블: 주차별 O/X + 미제출 수
  - 벌금 현황: 결석/미제출 1회당 7만원 계산
  - 최근 활동 요약

모든 파일은 한국어, Obsidian 마크다운, wikilink 사용.
