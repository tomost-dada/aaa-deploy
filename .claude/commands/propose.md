분석 데이터를 바탕으로 스킬 제안과 AAA 봇 인사이트를 업데이트해줘.

## 실행 절차

### 1. 데이터 수집
- `_분석/주차별분석/` 의 모든 분석 리포트 읽기
- `_분석/멤버별분석/` 의 모든 멤버 프로필 읽기
- `_제안/스킬_제안.md` 기존 제안 확인 (중복 방지)
- `_제안/AAA_봇_인사이트.md` 기존 내용 확인

### 2. 스킬 제안 도출
- **반복 작업 패턴** - 자동화할 수 있는 작업
- **공통 관심사 클러스터** - 3명 이상이 관심을 보인 주제
- **"해보고 싶은 것" 패턴** - 멤버들이 반복 언급한 실험
- **트렌드 연결** - 학습 내용과 AI/ML 트렌드 접점

제안 유형: Claude Code 스킬 / Obsidian 플러그인 / CLI 도구

### 3. 결과 업데이트
- `_제안/스킬_제안.md` 에 새 제안 추가 (난이도, 관련 개념, 추천 멤버)
- `_제안/AAA_봇_인사이트.md` 에 프로젝트 아이디어, 개선점, 자유 생각 추가

실현 가능한 것 위주, 1~2주 단위, 한국어, Obsidian wikilink 활용.
