# 통합 분석 스킬

주차별 미션, 개념, 인사이트를 통합 분석하고 현황을 업데이트합니다.

## 실행 절차

### 1. 데이터 수집
- `00_주차별미션/Week_0{{week}}/` 의 모든 멤버 파일 읽기
- `01_개념/` 의 모든 파일 읽기 (주제명만으로 된 플랫 구조, 전체 읽기)
- `02_인사이트/` 의 모든 파일 읽기 (주제명만으로 된 플랫 구조, 전체 읽기)
- 파일 크기 10바이트 이하인 파일은 미제출로 간주하고 건너뛰기

### 2. 제출 현황 집계
각 멤버(다다, 비비안, 띵끄, 잽, 흐민, 에밀리, 다니, 오웬)별로:
- 미션 제출 여부
- 개념 작성 여부
- 인사이트 작성 여부

### 3. 통합 분석
수집된 모든 내용을 종합하여:
- **주요 키워드** 추출 (빈도 순)
- **공통 관심사** 파악 (3명 이상 언급한 주제)
- **독특한 시각** 발견 (1명만 언급한 흥미로운 관점)
- **프로젝트 아이디어 시드** 도출 (멤버들의 "해보고 싶은 것"에서 추출)

### 4. 결과 파일 생성
- `_분석/주차별분석/Week_0{{week}}_분석.md` 생성
  - `_templates/주차별분석_템플릿.md` 형식을 따름
  - frontmatter에 제출 현황 데이터 포함
- 각 멤버별 `_분석/멤버별분석/{{닉네임}}_프로필.md` 업데이트 (누적)
  - 기존 파일이 있으면 이번 주 데이터를 추가
  - 없으면 새로 생성

### 5. 현황 업데이트
- `_현황/진행현황.md` 업데이트
  - **출석 현황** 테이블: 주차별 출석 O/X + 결석 수 집계
  - **과제 현황** 테이블: 주차별 과제 제출 O/미제출 X + 미제출 수 집계
  - **벌금 현황** 테이블: 출석 결석 또는 과제 미제출 1회당 7만원
    - 출석 벌금 = 결석 수 × 70,000원
    - 과제 벌금 = 미제출 수 × 70,000원
    - 누적 벌금 = 출석 벌금 + 과제 벌금
  - 최근 활동 요약

## 출력 형식
모든 파일은 한국어로 작성하고 Obsidian 마크다운 형식을 따릅니다.
wikilink(`[[파일명]]`)를 사용하여 관련 노트를 연결합니다.
