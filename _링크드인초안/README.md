# 링크드인 초안

> `/linkedin N` 실행하면 해당 주차 멤버별 링크드인 포스트 초안이 자동 생성됩니다.
> 초안을 복사해서 본인 스타일로 수정 후 링크드인에 게시하세요!

## 사용법

1. Claude Code에서 `/linkedin 2` 실행 (2주차 기준)
2. `_링크드인초안/Week_02_닉네임.md` 파일이 생성됨
3. 본인 파일을 열어서 내용 확인
4. 복사 → 링크드인에 붙여넣기 → 본인 스타일로 수정 → 게시
5. 게시 후 `_현황/진행현황.md`의 링크드인 포스팅 테이블에 링크 추가

## 파일 구조
```
_링크드인초안/
  Week_01_다다.md
  Week_01_비비안.md
  Week_02_비비안.md
  Week_02_띵끄.md
  ...
```
