# AI/ML 스터디 볼트

## 폴더 구조

### 멤버가 직접 쓰는 폴더
- 00_주차별미션/ : 주차별 과제 제출
- 01_개념/ : 학습 개념 정리
- 02_인사이트/ : 생각/의견/아이디어
- 03_회의/ : 주차별 회의록 + MVP 투표

### AI가 자동 생성하는 폴더 (건드리지 않아도 됨)
- _분석/ : 통합 분석 리포트
- _프로젝트제안/ : AI 프로젝트 제안
- _아카이브북/ : 학습 아카이브 북
- _현황/ : 진행 현황 대시보드

### 운영
- _meta/ : 운영 가이드 및 규칙
- _templates/ : 템플릿

## 과제 제출 방법
1. 본인 파일(Week_0N_닉네임.md) 열기
2. 내용 작성 후 저장
3. 터미널에서 `git add . && git commit -m "메시지" && git push origin main`
