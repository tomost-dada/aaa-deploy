# Week 01 미션 - Claude Code 입문

## 학습 내용
Claude Code의 기본 개념과 Agent Loop 구조를 이해했다. 사용자 입력 → 도구 호출 → 결과 반환의 사이클이 핵심이다.

## 실습 결과
- 설치 완료
- 파일 읽기/쓰기 테스트
- 마크다운 파일 자동 생성 실험

## 느낀 점
Agent Loop이 생각보다 단순하면서도 강력하다. 반복적인 작업을 자동화하는 데 큰 가능성이 보인다. 프롬프트 엔지니어링이 결과 품질에 큰 영향을 미친다.
