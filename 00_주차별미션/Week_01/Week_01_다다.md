# Week 01 미션 - Claude Code 입문

## 학습 내용
Claude Code CLI를 설치하고 기본 명령어를 익혔다. 터미널에서 `claude` 명령어로 바로 AI 어시스턴트를 사용할 수 있다는 점이 인상적이었다.

## 실습 결과
- Claude Code 설치 및 인증 완료
- 간단한 파일 생성/편집 테스트
- Git 연동 확인

## 느낀 점
GUI 없이 터미널만으로 AI와 협업할 수 있다는 것이 개발 워크플로우를 크게 바꿀 수 있겠다. 특히 Obsidian vault를 Claude Code로 관리하면 스터디 운영 자동화가 가능할 것 같다.
