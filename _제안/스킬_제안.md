## 상세 제안

### 제안 1: 원소스 멀티유즈 스킬 (`/multiuse`)
- **제안자**: AI (에밀리, 다니, 띵끄의 과제에서 도출)
- **유형**: Claude Code 스킬
- **난이도**: 중급
- **관련 개념**: 원소스 멀티유즈, 콘텐츠 자동화, 멀티채널
- **설명**: URL 하나를 입력하면 인스타 캐러셀 텍스트 + 스레드 포스트 + 링크드인 포스트 + 블로그 초안을 한 번에 생성. 에밀리는 8채널 20개 콘텐츠를, 다니는 3채널을 이미 구현 → 공통 스킬로 통합하면 모든 멤버가 사용 가능
- **기대 효과**: 콘텐츠 제작 시간 90% 단축
- **추천 멤버**: 에밀리, 다니, 띵끄
- **참고**: [[Week_02_에밀리]], [[Week_02_다니]], [[Week_02_띵끄]]

---

### 제안 2: 인스타 카드뉴스 생성 스킬 (`/cardnews`)
- **제안자**: AI (오웬의 과제에서 도출)
- **유형**: Claude Code 스킬
- **난이도**: 중급
- **관련 개념**: Playwright, HTML→PNG 변환, Claude API
- **설명**: 주제를 입력하면 5장 카드뉴스(커버→핵심→상세→요약→CTA) HTML을 생성하고 Playwright로 PNG 변환. 오웬이 crontab + instagrapi로 자동 업로드까지 구현한 것을 스킬로 패키징
- **기대 효과**: 카드뉴스 제작을 10분 이내로 단축
- **추천 멤버**: 오웬, 다니
- **참고**: [[Week_02_오웬]]

---

### 제안 3: CRM 알림톡 자동 생성 스킬 (`/crm`)
- **제안자**: AI (에밀리의 과제에서 도출)
- **유형**: Claude Code 스킬
- **난이도**: 고급
- **관련 개념**: n8n, SOLAPI, 에이전트 팀 구성, timeline-planner
- **설명**: 공유회/이벤트 URL을 입력하면 알림톡 9종 + 카플친 배너 텍스트 + 오픈채팅 카피를 자동 생성. 에밀리의 6개 에이전트 팀 구조(director→data-collector→timeline-planner→copywriter→media-ops→dispatcher)를 재사용
- **기대 효과**: CRM 세팅 작업을 수일 → 수시간으로 단축
- **추천 멤버**: 에밀리, 비비안
- **참고**: [[Week_02_에밀리]]

---

### 제안 4: CLAUDE.md 프로젝트 부트스트랩 스킬 (`/bootstrap`)
- **제안자**: AI (비비안, 에밀리의 과제에서 도출)
- **유형**: Claude Code 스킬
- **난이도**: 초급
- **관련 개념**: CLAUDE.md, PRD, 프로젝트 맥락 파일
- **설명**: 프로젝트 URL이나 설명을 입력하면 CLAUDE.md (프로젝트 맥락 파일)를 자동 생성. 비비안은 9단계에 걸쳐 기존 사이트를 분석한 후 CLAUDE.md를 작성했고, 에밀리도 PRD를 CLAUDE.md에 작성 → 이 과정을 스킬로 자동화
- **기대 효과**: 새 프로젝트 시작 시 컨텍스트 세팅 시간 대폭 단축
- **추천 멤버**: 비비안, 에밀리, 잽
- **참고**: [[Week_02_비비안]], [[Week_02_에밀리]]

### 제안 5: 웹 제안서 원클릭 생성기 (`/proposal`)
- **제안자**: AI (잽의 과제에서 도출)
- **유형**: Claude Code 스킬
- **난이도**: 중급
- **관련 개념**: HTML5 싱글파일, CSS 애니메이션, GitHub Pages 배포
- **설명**: 프로젝트 정보를 입력하면 프리미엄 원페이지 웹 제안서(HTML)를 생성하고 GitHub Pages로 자동 배포. 잽이 BRAND SHOW UP에서 증명한 "프레임워크 없이 순수 HTML/CSS/JS로 세련된 결과물" 패턴을 재사용
- **기대 효과**: PDF 제안서 대신 링크 하나로 클라이언트에게 전달
- **추천 멤버**: 잽, 띵끄
- **참고**: [[Week_02_잽]]

### 제안 6: Obsidian 호기심 트래커 플러그인
- **제안자**: AI (흐민의 과제에서 도출)
- **유형**: Obsidian 플러그인
- **난이도**: 고급
- **관련 개념**: Sullivan 프로젝트, 5개 Agent, 호기심 주도권
- **설명**: 흐민의 Sullivan 철학을 Obsidian 플러그인으로 구현. "형식 없이 던지는 생각 그대로 받고, 정리해주기 전에 질문 먼저" - 메모를 입력하면 AI가 답을 주는 대신 관련 질문을 던져서 생각을 확장. tone-guide 연동으로 개인화
- **기대 효과**: 피드 소비 → 자기 주도 학습으로 전환
- **추천 멤버**: 흐민, 다다
- **참고**: [[Week_02_흐민]]
